Class Selector

Elements with the same class can share styles. This paragraph uses the .highlight class.

ID Selector

This section is styled using an #id selector. IDs are unique for one element.

Descendant Selector

Only paragraphs inside .container are affected by the descendant selector.

This paragraph is outside the container, so it is not affected.

Group Selector

Group selectors allow you to apply the same style to multiple elements at once.
